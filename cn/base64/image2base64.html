使用示例

CSS 背景图片

.background {
    background-image: url('data:image/jpeg;base64,/9j/4AAQSkZJRgABA...');
    background-size: cover;
}

JavaScript 图片

const img = new Image();
img.src = 'data:image/jpeg;base64,/9j/4AAQSkZJRgABA...';
document.body.appendChild(img);

JSON 数据

{
    "image": "data:image/jpeg;base64,/9j/4AAQSkZJRgABA...",
    "type": "jpeg",
    "size": "1024x768"
}

注意事项

  • 文件大小：Base64 编码会增加约 33% 的文件大小，大图片可能影响页面加载速度
  • 浏览器缓存：内嵌图片无法被浏览器单独缓存，可能影响性能
  • 代码长度：生成的 Base64 字符串很长，建议用于小图标或重要图片
  • 兼容性：所有现代浏览器都支持 Base64 图片，IE8+ 也支持
  • 隐私安全：图片内容完全可见，不适合存储敏感信息
